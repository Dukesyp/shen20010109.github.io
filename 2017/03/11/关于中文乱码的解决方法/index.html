关于中文乱码的解决方法

开始的时候，博客中凡是涉及到汉字的部分全部显示为乱码，多方询问无解
后来经许多dalao的指点，终于知道需要转码为UTF-8，又在网上看到了网友们推荐的Sublime Text

关于Sublime Text

这个软件可以直接把文件存为UTF-8，非常方便，但是如果你手残不小心点到窗口右上角的“×”，并不弹出保存文件的提示……
此软件是需要付费的，可以去网上找一下破解版。
