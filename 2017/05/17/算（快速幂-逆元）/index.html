算（快速幂+逆元）

题目

【问题背景】

zhx 帮他妹子做数学题。

【问题描述】

求：

如 N=3, M=3，这个值为 1^1+1^2+1^3+2^1+2^2+2^3+3^1+3^2+3^3=56。

【输入格式】

仅一行，包含两个数 N 和 M。

【输出格式】

仅一行，包含所求的答案 mod 10^9 + 7 的值。

【样例输入】

3 3

【样例输出】

56

【数据范围与规定】

50%的数据，所有1 ≤ $ m,n $ ≤ 10。
对于100%的数据，所有1 ≤ $ m,n $ ≤ 50000。

这个题暴力的话，很显然是过不了100%的数据的，所以就要考虑其他方法
仔细分析一下的话，很容易就会发现：这就是等比数列求和的问题啊，复杂度瞬间从O(mn)降到了O(mlg n)

特别要注意的是优先级，在运算中取模运算是优先的，所以要加括号
再就是要用逆元……

逆元：$$ x^{p-1} \equiv 1 \, (mod \, p) $$
